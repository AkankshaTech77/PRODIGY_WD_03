 # 🎮 Tic Tac Toe Web Application

## Task-03 | Prodigy InfoTech Web Development Internship

A responsive and interactive Tic Tac Toe Web Application developed using HTML, CSS, and JavaScript. The game allows two players to compete against each other while automatically detecting winners, draws, and providing an option to restart the game.

---

## 🚀 Features

* 🎯 Interactive 3×3 Tic Tac Toe Board
* 👥 Two-Player Gameplay (X and O)
* 🏆 Automatic Winner Detection
* 🤝 Draw Detection
* 🔄 Restart Game Functionality
* 📱 Responsive User Interface

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Project Objective

The objective of this project is to build an interactive Tic Tac Toe game that:

* Allows two players to play alternately
* Tracks and updates the game state
* Detects winning combinations
* Identifies draw situations
* Provides a seamless user experience
